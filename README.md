# 🎮 Hangman Game using Python

## **📌 Project Overview**

The **Hangman Game** is a simple console-based Python project where the
player attempts to guess a hidden word by entering one letter at a time.

The system randomly selects a word from a predefined list, and the
player must guess the correct letters within a limited number of
incorrect attempts (6 chances).

This project helps beginners understand fundamental Python programming
concepts such as loops, conditions, lists, user input handling, and the
use of modules.

## **🕹️ Game Instructions**

-   The system selects a random word.\
-   The word is displayed using underscores (\_).\
-   Guess one letter at a time.\
-   Each wrong guess reduces your attempt count.\
-   You are allowed only **6 incorrect guesses**.\
-   The game ends when:
    -   You guess the word correctly (**You Win**), or\
    -   All attempts are used (**Game Over**)
