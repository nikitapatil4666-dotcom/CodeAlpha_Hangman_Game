##🎮 Hangman Game – Python Project
##📌 Project Title

Text-Based Hangman Game using Python

##📖 Project Description

This project is a simple text-based implementation of the classic Hangman Game developed using Python programming language.
In this game, the player has to guess a hidden word by entering one letter at a time within a limited number of attempts.

The system randomly selects a word from a predefined list, and the player must guess the correct letters before the maximum number of incorrect guesses (6 attempts) is reached.

##🎯 Objective

The main objective of this project is to:

Understand basic Python programming concepts

Implement loops and conditional statements

Use lists and string operations

Apply the random module for word selection

Develop logic-building skills through game development

##🛠️ Technologies Used

Python

Random Module

Console-based Input/Output

##🧩 Features

Predefined list of 5 words

Random word selection

Letter-by-letter guessing

Maximum 6 incorrect attempts allowed

Displays correctly guessed letters

Notifies repeated guesses

Win/Loss result display at the end

##▶️ How to Run the Program

Follow these steps to run the game:

Install Python on your system.

Open any Python IDE (e.g., IDLE / VS Code / PyCharm).

Copy the Hangman game code into a new Python file.

Save the file as:

hangman_game.py

Run the program.

Enter one letter at a time to guess the hidden word.

##🕹️ Game Instructions

The system will choose a random word.

The word will be displayed as underscores (_).

You must guess the correct letters.

Each incorrect guess reduces your attempt count.

You are allowed only 6 wrong attempts.

The game ends when:

You guess the word correctly (Win), or

You run out of attempts (Lose)
